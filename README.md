# 🌸 Animated Flowers Scene

## Description
Interactive 3D flower and plant scene with lighting and movement effects using only HTML and CSS.  
Flowers and grass grow and sway with smooth animations, set against a night-themed background.

## Features
- 3D flower animations
- Growing grass with animated leaves
- Night background with light effects
- Fully CSS-based animations
